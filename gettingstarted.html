Support for Existing Cross-Tabs in Excel Format

If you have existing cross-tab output from Wincross or SPSS with each table in a separate sheet, simply use the Cross Tab Transformer from the top navigation menu. Upload your file, and Ragenaizer will convert it into a JSON format, which you can then download to your PC and upload for data analysis.

File Format for Data Analytics
